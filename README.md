# Simple Python To-Do List CLI App

A minimal, interactive Command Line Interface (CLI) To-Do List application written in Python. This project allows users to add multiple tasks dynamically and displays the final list upon exit.

## 🚀 Features
* **Dynamic Input:** Add as many tasks as you want.
* **Easy Exit:** Stop adding tasks anytime by typing 'n'.
* **Clean Output:** Displays a numbered list of all your tasks at the end.

## 📋 Code Overview
The application uses a basic `while` loop, a Python list to store tasks, and `enumerate()` to format the final output neatly.

## 💻 How to Run
1. Make sure you have Python installed.
2. Clone this repository:
   ```bash
   git clone https://github.com
   ```
3. Navigate to the project folder:
   ```bash
   cd python-todo-list-cli
   ```
4. Run the script:
   ```bash
   python todo.py
   ```

## 🛠️ Requirements
* Python 3.x
